# 🍽️ Zomato Restaurant Data Analysis

## 📌 Project Overview

This project analyzes the Zomato Restaurants Dataset using Python, SQL, and Power BI. The dataset was cleaned and prepared using Python, explored with SQL queries, and visualized through an interactive Power BI dashboard to gain meaningful business insights.

---

## 📂 Dataset Selected

**Zomato Restaurants Dataset**

---

## 🎯 Why I Selected This Dataset

I selected the Zomato Restaurants Dataset because it contains useful information such as restaurant names, restaurant types, cuisines, ratings, average cost, online ordering, and locations. It is suitable for practicing data cleaning, SQL analysis, and dashboard creation.

---

## ❓ Business Questions Answered

1. Which areas have the highest number of restaurants?
2. Which restaurant types have the highest number of restaurants?
3. Does the average cost for two people have any relationship with restaurant ratings?

---

## 💡 Key Insights

- The dataset contains **7,106 restaurants**.
- A few areas have significantly more restaurants than others.
- Quick-service and casual dining restaurants make up a large portion of the dataset.
- Online ordering is widely supported by restaurants.
- Higher average cost does not always correspond to higher customer ratings.
